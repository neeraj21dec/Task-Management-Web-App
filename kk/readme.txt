Here we will see what softwares i used to run and host my java website

java website= "kk infotech"

Step 1).First of we will make our java website with the help of html,css,javascript,mdbootstrap in dreamweaver.

Step 2).Then we convert all the html pages to jsp(java) pages and connect with database using mysql connector.


step 3).Then we will deploy our website in our local server "Apache Tomcat 8080" and run on it by
 typing address "Localhost/8080/ourwebsitefirstpage.jsp".
   

Step 4).For hosting we use ngrok server by typing command in commandlineprompt "ngrok.exe http 8080"

      Here we will get our url for our website.


Software Requirements :-

1.Adobe Dreamweaver/notepad
2.mdbootstrap package
3.tomcat server
4.database= MYSQL
5.mysqlconnector.jar
6.ngrokserver.exe
